# Number Nonsense

A fast, intuitive, and competition‑ready training platform for mastering UIL Number Sense curveballs, strategies, and timed drills.

Built by **Noah Taylor**  
Stony Point High School — UIL Number Sense Team 2026

---

## 🚀 Features

### **Practice Mode**
- Auto‑generated curveball problems  
- Instant feedback  
- Auto‑advance on correct answers  
- Mixed‑mode randomization  

### **Timed Mode**
- 2‑minute competitive drill  
- Select specific curveball types  
- Score tracking  
- Smooth, contest‑style flow  

### **Explanations Tab**
- Strategy breakdowns  
- Curveball logic  
- UIL‑style shortcuts  
- Clean MathJax formatting  

### **Stylized Intro Page**
- Title, credits, and mission statement  
- Clean, modern UI  

---

## 🧮 Curveball Types Included
- Fraction traps  
- Permutation/inequality counts  
- Digit‑sum combinatorics  
- Product notation (∏)  
- nPr / nCr relationships
